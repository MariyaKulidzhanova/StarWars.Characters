# StarWars.Characters

Тестовое задание на вакансию "fullstack .net developer".

_________________________________________________
Суть задания:
Необходимо создать мини-сайт, который представляет из себя каталог персонажей вселенной Star Wars. 
Примеры макетов форм дополнительно приложены. 

На сайте должны быть страницы:
- список/каталог персонажей (с фильтрацией/пагинацией);
- страница карточки персонажа;
- страница редактирования данных о персонаже;
- страница добавления нового персонажа.

Для каждого персонажа хранятся: 
- имя, дата рождения, планета происхождения, пол, раса, рост, цвет волос, цвет глаз, описание, фильмы в которых задействован.

_________________________________________________
Технические требования:
- Ко внешнему виду требования минимальные, но всё должно выглядеть аккуратно и приятно. 
- На бекенде должны быть ASP.NET Core MVC и Entity Framework Core (например на основе .NET 6)
- SQLite (для упрощения развёртывания без необходимости устанавливать полноценные СУБД типа MSSQL/PostgreSQL - скачал проект и запустил)
- На фронтенде можно использовать jQuery & Bootstrap.
В остальном допустимо использовать любые сторонние библиотеки, в том числе nuget-пакеты. Например, Automapper.

_________________________________________________

Примечание:
Опционально. Аля "программа-максимум" при вашем желании (будет плюсом в карму).
- Подключить механизмы регистрации/аутентификации/авторизации, например посредством ASP.NET Core Identity. При этом, учесть что в таком случае редактировать персонажа будет иметь право только тот, кто изначально выложил информацию о нем.
